## 作った背景
端末の内部ストレージのパスと、外部ストレージ（SDカード）のパスって、Androidの機種やOSによっていろいろ変わるので、簡単にわかるようにしようと思って、作りました。
知りたいと思った時にパッと出せなかったのがストレスだったのです。

あと、とある調査で、余計なファイルやコードが入っています。
